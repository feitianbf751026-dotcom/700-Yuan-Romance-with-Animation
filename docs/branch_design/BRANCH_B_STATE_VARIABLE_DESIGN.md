# Branch B 状态变量设计

## 1) Branch B 核心主题总结

基于关键场景证据，Branch B 的核心主题是：**从发消息道歉开始，通过聊天和见面逐渐建立信任，经历冲突和坦白，最终建立关系**。证据包括：SCENE_1B_MESSAGE 中她回复"没关系的"并询问"你还好吗"；SCENE_1B_CHAT 中她透露"我也很庆幸你没来"；SCENE_1B_9C_FULL_CONFESSION 中她表示理解并透露"我也一直在观察你"；SCENE_1B_10A_RELATIONSHIP_GROWTH 中她评价主角"太善良了"。

## 2) 状态变量选择

**选择**: `trust_level (0..3)` 关系信任/接受度

**理由**: Branch B 的核心是双方从陌生到建立信任的过程，从发消息道歉、聊天、见面、冲突、坦白的每个阶段都围绕信任的建立和维持，最终关系的发展取决于信任程度。

## 4) 呈现差异点（Scene -> trust_level 分段表现）

| Scene 常量名 | trust_level 0-1（低信任） | trust_level 2-3（高信任） |
|-------------|------------------------|------------------------|
| SCENE_1B_CHAT | 她回复较冷淡，询问"你还好吗"更像是职业话术，对话简短，没有深入交流。 | 她回复温暖，主动分享自己的故事（"我也很庆幸你没来"），对话深入，建立情感连接。 |
| SCENE_1B_COFFEE | 她准时出现但保持距离，对话礼貌但疏远，没有深入交流，主角解释时她只是礼貌地点头。 | 她准时出现且主动交流，对话自然亲切，主角解释时她表示理解并分享自己的感受，建立信任。 |
| SCENE_1B_4C_HESITATION | 她察觉到主角的心事，但只是礼貌地询问，没有深入追问，对话保持距离。 | 她察觉到主角的心事，主动关心并询问，表现出理解和耐心，鼓励主角坦诚。 |
| SCENE_1B_6C_CONFLICT | 冲突后她认为"我们真的不适合"，态度坚决，没有和解的意愿，关系可能结束。 | 冲突后她虽然也认为"不适合"，但语气中有犹豫，愿意尝试理解，有和解的可能。 |
| SCENE_1B_7B_CONFESSION_MOMENT | 主角坦白后，她只是礼貌地表示理解，没有深入交流，对话保持距离，没有推进关系的意愿。 | 主角坦白后，她表示理解并分享自己的感受（"我也一直在想"），对话深入，表现出推进关系的意愿。 |
| SCENE_1B_9A_FIRST_DATE | 她准时出现但保持距离，对话礼貌但疏远，没有深入交流，主角感到紧张和不安。 | 她准时出现且主动交流，对话自然亲切，主角感到放松和温暖，两人关系进展顺利。 |
| SCENE_1B_9C_FULL_CONFESSION | 主角坦白后，她只是礼貌地表示理解，没有深入交流，对话保持距离，没有推进关系的意愿。 | 主角坦白后，她表示理解并透露"我也一直在观察你"，对话深入，表现出推进关系的强烈意愿。 |
| SCENE_1B_10A_RELATIONSHIP_GROWTH | 两人开始交往但关系进展缓慢，她评价主角"太善良"但语气平淡，没有深入的情感连接。 | 两人开始交往且关系进展顺利，她评价主角"太善良"时语气温暖坚定，表现出深入的情感连接和信任。 |
