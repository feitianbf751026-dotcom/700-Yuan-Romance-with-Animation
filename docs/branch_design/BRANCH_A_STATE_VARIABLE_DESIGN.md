# Branch A 状态变量设计

## 1) Branch A 核心主题总结

基于关键 scene 证据：
- **观察阶段**：主角不敢上前，选择观察和跟踪
- **初遇阶段**：鼓起勇气主动打招呼，她记得那晚的事
- **建立信任**：经常见面，互相了解，她评价"太善良"
- **冲突转折**：跟踪被发现，主角决定坦白一切
- **错过结局**：三个月后回去找她，发现她已有新男友

**核心主题**：主角因**勇气/主动性不足**而错过机会。从"不敢上前"到"跟踪被发现"再到"坦白"，最终因行动太晚而错过。这是一个关于**勇气与时机**的故事。

## 2) 选择的状态变量

**选择**: `courage_level` (0..3) - 行动勇气/主动性

**理由**：Branch A 的核心冲突是主角的"不敢"与"错过"，从观察、跟踪到最终坦白，所有关键节点都围绕"是否有勇气主动行动"展开。`courage_level` 直接反映主角的行动意愿，影响是否能抓住机会。
