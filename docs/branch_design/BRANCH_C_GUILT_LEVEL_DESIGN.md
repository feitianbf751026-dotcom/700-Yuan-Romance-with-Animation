# Branch C guilt_level 设计规则

## 1) guilt_level 定义

**含义**: 主角对700块钱和逃避行为的内疚程度，影响后续对话和选择的呈现。  
**取值范围**: 0-3（整数）
- 0: 无内疚/已释怀
- 1: 轻微内疚/偶尔想起
- 2: 中度内疚/经常困扰
- 3: 重度内疚/持续折磨

## 2) 选择触发点（lastChoice -> guilt_level 变化）

| lastChoice | guilt_level 变化 | 理由 |
|------------|------------------|------|
| 1c_1 | +1 | 选择逃避/删除联系方式，内疚增加 |
| 1c_2 | -1 (clamp 0..3) | 主动发送第一次消息，内疚减轻 |
| 1c_3 | +1 | 选择跟踪/观察，内疚增加 |
| 1c_2b | +1 | 继续观察而非直接接触，内疚加深 |
| 1c_3a | -1 (clamp 0..3) | 主动道歉，内疚减轻 |
| 1c_4c_a | -1 (clamp 0..3) | 寻求原谅，主动面对，内疚减轻 |
| 1c_4c_b | +1 | 继续逃避/接受现实，内疚增加 |
| 1c_4c_c | +1 | 继续观察/内疚加深路径，内疚增加 |
| 1c_4d_a | -1 (clamp 0..3) | 主动约第一次见面，内疚减轻 |
| 1c_5c_a | -1 (clamp 0..3) | 突破障碍，主动沟通，内疚减轻 |
| 1c_5c_b | +1 | 遇到障碍/保持距离，内疚增加 |
| 1c_7b_a | 设为 1 | 坦白时刻，承认过去，内疚大幅减轻 |
| 1c_7b_c | +1 | 保持距离/继续逃避，内疚增加 |
| 1c_7g_a | 设为 0 | 选择救赎路径，完全释怀 |

## 3) 呈现差异点（Scene -> guilt_level 分段表现）

| Scene 常量名 | guilt_level 分段 | 表现差异 |
|--------------|-------------------|----------|
| SCENE_1C_GUILT | 0-1 vs 2-3 | 0-1: 轻微提及，主角能正常思考；2-3: 强烈内疚，影响睡眠和日常 |
| SCENE_1C_4C_GUILT_DEEPEN | 0-1 vs 2-3 | 0-1: 偶尔想起，不影响决策；2-3: 持续困扰，做噩梦，影响判断 |
| SCENE_1C_4D_SEEK_FORGIVE | 0-1 vs 2-3 | 0-1: 轻松道歉，语气自然；2-3: 沉重道歉，充满自责和痛苦 |
| SCENE_1C_5C_BARRIER | 0-1 vs 2-3 | 0-1: 障碍来自外部/误解，容易沟通；2-3: 障碍来自内疚，难以开口 |
| SCENE_1C_5D_BREAKTHROUGH | 0-1 vs 2-3 | 0-1: 突破轻松，关系自然发展；2-3: 突破艰难，需要克服内疚才能开口 |
| SCENE_1C_7B_CONFESSION_MOMENT | 0-1 vs 2-3 | 0-1: 坦白轻松，像分享故事；2-3: 坦白沉重，每个字都像从心底挖出 |
| SCENE_1C_8D_PAST_CONFESS | 0-1 vs 2-3 | 0-1: 回顾过去时语气平静；2-3: 回顾过去时充满痛苦和自责 |
